# Local Himudo videos

Drop your own `.mp4` / `.webm` files in this folder, then list them in `manifest.json`.

## Example `manifest.json`

```json
[
  {
    "file": "promo-galon.mp4",
    "title": "Promo Galon Himudo",
    "desc": "Penawaran antar gratis Medan"
  },
  {
    "file": "proses-pabrik.mp4",
    "title": "Proses Pabrik",
    "desc": "10 tahap pencucian galon"
  }
]
```

The homepage video section loads these automatically as extra cards.

Official YouTube videos are already embedded on the site.
